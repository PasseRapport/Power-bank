# Power-bank

The objective of this project is to upcycle an old flashlight into a usb C power bank while conserving the original look and functionnality.

It is based on the IP2368 charge controller wich allow for dual mode Power Delivery charging up to 100W. However this board is designed to to supply 45w max (5 Amps at 9 Volts).
